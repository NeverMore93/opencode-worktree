# CLAUDE.md

## Responsibility Boundary
- This directory owns the canonical document templates used by the SpecKit workflow.
- Templates here define structure and wording conventions, not feature-specific content.

## Allowed Content
- Spec, plan, tasks, checklist, constitution, and agent templates
- Template-level guidance comments
- Small supporting template assets that are shared across workflow runs

## Forbidden Content
- Filled-in feature documents
- Runtime scripts or implementation code
- Repository-specific requirements that should live in generated artifacts under `specs/`
