# CLAUDE.md

## Responsibility Boundary
- This directory owns executable workflow automation used by SpecKit in this repository.
- It is for automation entrypoints and shared script scaffolding, not for templates or design content.

## Allowed Content
- Script subdirectories grouped by runtime or shell
- Shared automation helpers
- Repo workflow scripts that support feature/spec generation

## Forbidden Content
- Template markdown files
- Feature-specific specs or checklists
- Product source code or terminal-only notes without executable value
