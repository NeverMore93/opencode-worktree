# CLAUDE.md

## Responsibility Boundary
- This directory owns repo-local notes about which external skills and subagent patterns work well for this repository.
- It documents workflow recommendations only; it is not the source of truth for shared skill bodies.

## Allowed Content
- Skill selection guides and workflow maps
- Notes on recommended subagent patterns for repo tasks
- Lightweight indexes that point to external or user-scoped skill sources

## Forbidden Content
- Full copies of shared skill implementations
- Product source code or feature specs
- One-off session notes that belong under `cases/`
