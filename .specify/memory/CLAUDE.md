# CLAUDE.md

## Responsibility Boundary
- This directory owns stable project memory used to keep repository-wide governance and context aligned across sessions.
- Files here may summarize current code truth or active design state, but they do not replace feature-local specs.

## Allowed Content
- `constitution.md`
- Stable project memory and context summaries that apply across sessions
- Small supporting notes that point readers to canonical feature-local docs

## Forbidden Content
- Product source code or implementation scripts
- Feature-local specs, plans, tasks, or checklists that belong under `specs/`
- Large duplicated copies of design documents that should remain canonical elsewhere
