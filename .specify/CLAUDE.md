# CLAUDE.md

## Responsibility Boundary
- This directory owns the repository's SpecKit engine assets, stable memory, templates, scripts, and tool integrations.
- It defines how specification-driven workflows run in this repo; it does not own product source code or feature content.

## Allowed Content
- Canonical SpecKit templates
- Workflow automation scripts
- Host/editor integration adapters for SpecKit
- Stable project memory such as the constitution and project-wide context summaries

## Forbidden Content
- Plugin implementation code
- Feature-specific specs, plans, tasks, or checklists
- Repository documentation unrelated to the SpecKit workflow
